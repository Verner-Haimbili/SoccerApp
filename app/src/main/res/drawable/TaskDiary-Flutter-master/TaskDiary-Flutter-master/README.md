# Task Diary-App-Flutter

Task Manager App Made in Flutter.
 
#### Supported features:
- Create task 
- Dashboard
- Custom Task Label 
- Offline Storage in Hive DB

Test App by Downloading Apk From Screenshot/app.apk

#### Getting Started:

1. Clone/Download The Repo 
4. run ```flutter run ```
